# ATM Machine (Core Java)

## Description

A simple console-based ATM Machine application developed using Core Java.

## Features

- Check Balance
- Deposit Money
- Withdraw Money
- Exit Menu
- Menu-driven interface

## Technologies Used

- Java
- IntelliJ IDEA

## Concepts Used

- Object-Oriented Programming (OOP)
- Classes and Objects
- Methods
- Loops
- switch-case
- Scanner
- Refactoring

## Future Improvements

- PIN Authentication
- Multiple User Accounts
- File Handling
- Exception Handling
- Database Integration
